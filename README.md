# Rails App with Google SSO Authentication

This is a Ruby on Rails application that uses **Devise** and **OmniAuth** for Single Sign-On (SSO) with **Google**.

---

## 🛠 Requirements

- Ruby >= 3.0
- Rails >= 7.0
- PostgreSQL (or any database of your choice)
- Google Developer Account (for OAuth credentials)
